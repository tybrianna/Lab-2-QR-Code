## Overview
The goal of this project was to generate an HTML and CSS file to replicate the QR Code given in Frontend Mentor. I was given a template to go off of and to go from there to have it match up as best as possible.



### Links

- Fronten Mentor URL(https://www.frontendmentor.io/challenges/qr-code-component-iux_sIO_H)

## My process
- I started off by looking at the HTML that was provided to me as a guide to start. I noticed  that based on the original image I would need to create a flexbox. So, within the html I added the QR code image and placed that and the body inside a containter to customize in CSS. Once those were add I then linked my html to my css to begin the customization process. The make my image match the original I have to adjust the border radius for both the image and flexbox. I also had to match up all the colors (font and background colors). Once that was complete I noticed that the image inside the flexbox did not align perfectly like in the original. After playing around with the placements, height, and width, I was finally able to get it almost perfect. Once all was complete, I then pushed it to my Github Repository.
